# cold_install

一键安装各种冷门科学上网协议脚本，支持TUIC、shadowsocks(2022)

```shell
curl -O https://raw.githubusercontent.com/tdjnodj/cold_install/main/cold_install.sh && bash cold_install.sh
```

# 提示
需要依赖本项目的api，所以不建议fork使用！
